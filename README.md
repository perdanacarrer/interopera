# InterOpera — Compliance Report Engine

## Single command (Docker — recommended)

```bash
docker compose up
```

Runs both Firm A and Firm B. Output XLSX written to `reports/`.

**With Gemini narrative** (free key at https://aistudio.google.com/app/apikey):
```bash
GEMINI_API_KEY=your_key_here docker compose up
```

If `GEMINI_API_KEY` is not set, narrative is skipped automatically — all 5 constraints still hold.

---

## Without Docker (Mac / Linux)

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt

python main.py --firm firm_A
python main.py --firm firm_B
```

Or use the one-liner:
```bash
bash setup_and_run.sh firm_A
bash setup_and_run.sh firm_B
```

## Without Docker (Windows)

```bash
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt

python main.py --firm firm_A
python main.py --firm firm_B
```

## Narrative (optional, free)

Narrative commentary is generated by **Google Gemini** (free tier, no credit card needed):

1. Get a free key: https://aistudio.google.com/app/apikey
2. Set it:
   ```bash
   export GEMINI_API_KEY=your_key_here        # Mac/Linux
   set GEMINI_API_KEY=your_key_here           # Windows
   GEMINI_API_KEY=your_key docker compose up  # Docker
   ```

Narrative is **auto-enabled** when the key is present, **auto-skipped** when absent.

## The Five Constraints

| # | Constraint | How satisfied |
|---|---|---|
| 1 | **Reproducible** | Pure Python deterministic arithmetic on fixed inputs |
| 2 | **Traceable** | Every figure has `graph_path` + `citation`. Untraceable → `ERROR`, never silent |
| 3 | **No LLM numbers** | LLM called only in `narrative.py` after compute; receives only final figures; firewall verified |
| 4 | **Firm A reconciliation** | 13/13 figures match `firm_A_answer_key.xlsx` exactly |
| 5 | **Firm B reconfiguration** | 3 JSON flags in `config/firm_B.json`; zero engine code changes |
