# InterOpera — Compliance Report Engine

Senior Software Engineer Take-Home Assessment submission.

## Quick Start

### Single command (Docker — recommended)

```bash
docker compose up
```

This runs **both** Firm A and Firm B reports in sequence. Output XLSX files are written to `reports/`.

To run firms individually:
```bash
docker compose run firm_a   # Firm A only
docker compose run firm_b   # Firm B only
```

With Gemini narrative (optional — free key at https://aistudio.google.com/app/apikey):
```bash
GEMINI_API_KEY=your_key_here docker compose up
```

### Without Docker (Mac / Linux)

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt

python main.py --firm firm_A --no-narrative
python main.py --firm firm_B --no-narrative
```

Or use the one-liner setup script:
```bash
bash setup_and_run.sh firm_A
bash setup_and_run.sh firm_B
```

### Without Docker (Windows)

```bash
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt

python main.py --firm firm_A --no-narrative
python main.py --firm firm_B --no-narrative
```

**No code changes between firm runs.** Switching `--firm firm_B` loads `config/firm_B.json` which drives all three convention differences.

## LLM API key (optional)

Narrative commentary is generated by **Google Gemini** (free tier). To enable it:

1. Get a free key at https://aistudio.google.com/app/apikey (no credit card needed)
2. Run:

```bash
# Docker
GEMINI_API_KEY=your_key_here docker compose up

# Without Docker
export GEMINI_API_KEY=your_key_here
python main.py --firm firm_A
```

Without the key, narrative is skipped and all 5 constraints still hold (the LLM is only in the narrative path, never in the computation path).

## The Five Constraints

| # | Constraint | How satisfied |
|---|---|---|
| 1 | **Reproducible** | All figures computed by deterministic Python arithmetic on fixed CSV/PDF inputs. Same inputs → byte-identical figures. |
| 2 | **Traceable** | Every FigureResult carries `graph_path` (explicit NetworkX node/edge traversal) + `citation` (source_doc, page, chunk_id, passage). Figures that cannot be traced return `"status": "ERROR"`. |
| 3 | **No LLM numbers** | LLM is invoked only in `narrative.py`, after compute is complete, receives only final figures. Firewall check verifies no novel numbers appear in narrative. |
| 4 | **Firm A reconciliation** | 13/13 figures match `firm_A_answer_key.xlsx` exactly. Run `python main.py --firm firm_A` to verify. |
| 5 | **Reconfigurable to Firm B** | Three boolean/string flags in `config/firm_B.json` produce all three Firm B differences. Zero engine code changes. |

## Audit Log

`audit/audit_log.db` — SQLite with append-only guarantee:

```sql
-- These triggers are active in the database:
CREATE TRIGGER prevent_update BEFORE UPDATE ON audit_log
  BEGIN SELECT RAISE(ABORT, 'audit_log is append-only: UPDATE is prohibited'); END;
CREATE TRIGGER prevent_delete BEFORE DELETE ON audit_log
  BEGIN SELECT RAISE(ABORT, 'audit_log is append-only: DELETE is prohibited'); END;
```

Events logged: `CONFIG_LOADED`, `GRAPH_BUILT`, `FIGURES_COMPUTED`, `NARRATIVE_GENERATED`, `REPORT_WRITTEN`, `RECONCILIATION_COMPLETE`, `RUN_COMPLETE`.
